ZEK-LIBRARY

The key feature of ZEK-Library is its ability to download journals and books for free, allowing users to access knowledge without having to pay.

Built using C# as the frontend and Python as the backend, ZEK-Library combines efficient performance with an attractive user interface. The download process is done asynchronously, enabling users to download multiple files simultaneously without any hindrance.
